# MICEkart — Static Frontend (Vite + React)

This repository is a small static React site scaffold for MICEkart with Tailwind CSS, Framer Motion and Lucide icons.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Build

```bash
npm run build
npm run preview
```

Notes

- The contact form posts to `https://formsubmit.co/contact@micekart.com` as a placeholder — replace with the correct email or configure Web3Forms.
- To deploy, build and push `dist/` to any static hosting (Vercel, Netlify, GitHub Pages).
